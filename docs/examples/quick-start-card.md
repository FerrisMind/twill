# Quick Start Card

This is the shortest possible Twill example for `main`/`0.3.x`.

- File: `examples/00_quick_start_card.rs`
- Run:

```bash
cargo run --example quick_start_card
```

- Expected output: a tiny terminal dump showing a reusable card style and a button style with hover and focus-visible state layers.

Why this exists:
it gives users a 30-second copy-paste success before they learn the deeper builder and composition patterns.
